# @strato-admin/create

The official scaffolding tool for Strato Admin projects.

## Usage

```bash
pnpm create @strato-admin
```

This will guide you through creating a new Strato Admin project with a standard configuration (Vite + React + Cloudscape).

## Development

To test the CLI locally:

```bash
pnpm build
node dist/index.js my-test-project
```
